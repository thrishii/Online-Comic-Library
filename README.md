# Online-Comic-Library
Author
Name: Thrishika R

Project Description
This project presents the system design of an Online Comic Library System using SysML diagrams.
The diagrams illustrate system functionality, structure, and internal interactions.

Use Case Diagram
This diagram shows interactions between actors (User and Admin) and the system functionalities such as sign up, login, browsing comics, searching, and managing comics.

Block Definition Diagram (BDD)

This diagram represents the system structure with blocks such as User, Admin, Comic, ComicLibrary, Authentication, and Search, including their attributes.

Internal Block Diagram (IBD)

This diagram shows how internal components like userManagement, comicManagement, searchSystem, and authenticationSystem interact within the system.
